Resumen Tema 2

¿Qué es JavaScript?

JavaScript es un lenguaje interpretado, débilmente tipado, basado en objetos y funciones, con clases basadas en prototipos y usado en cliente-servidor y escritorio. Adopta la nomenclatura lowerCamelCase.

Sintaxis

El código JavaScript puede incluirse mediante la etiqueta <script>, en eventos, URL con pseudo-protocolo javascript, o como archivo externo.

Variables y Constantes

JavaScript incluye let para ámbito local, var para global, y const para constantes. Se debe usar "use strict" para modo estricto.

Tipos de Datos

Los tipos de datos incluyen primitivos (boolean, number, string, undefined) y objetos (como función y object). También se puede usar typeof para identificar tipos.

Operadores y Estructuras de Control

JavaScript cuenta con operadores de asignación, comparación, aritméticos y lógicos, y estructuras de control alternativas e iterativas.

Funciones

Existen funciones declarativas, anónimas y de flecha. Las funciones son objetos, permiten parámetros por defecto y admiten pasar funciones como argumentos.

JSON

JavaScript Object Notation es un formato para estructurar datos. Se puede convertir a texto con JSON.stringify y a objeto con JSON.parse.

Operador Spread

El operador spread (...) permite expandir elementos de un array u objeto. Es útil para combinar arrays o copiar objetos sin modificar el original.

Ejemplo:

let numeros = [1, 2, 3]; let copia = [...numeros];

Objetos Predefinidos

JavaScript cuenta con varios objetos predefinidos, como Number, String, Date, Math, y Array, que ofrecen métodos y propiedades para manipular datos y realizar cálculos.

Ejemplo:

Math.sqrt(16); // Devuelve 4
